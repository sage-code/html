Learning HTML

HTML is a document oriented language and looks like XML. Both languages are markup languages based on tags. These are keywords enclosed in angle brackets: One opening tag looks like this: <tag>. Each opening tag has a closing tag that look like this: </tag>. Each tag has specific attributes and functionality. There are about 30 tags to learn before you can say you know HTML.
